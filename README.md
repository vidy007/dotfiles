# List of programs
* File manager - [ranger](https://github.com/ranger/ranger) (soon will be replaced with [yazi](https://github.com/sxyazi/yazi))
* ~~Text editor~~ IDE - [vim](https://github.com/vim/vim)
* Compositor - [Pijulius' picom fork](https://github.com/pijulius/picom)
* WM - [i3](https://github.com/i3/i3)
* Terminal - [kitty](https://github.com/kovidgoyal/kitty)
* System fetch - [nitch](https://github.com/ssleert/nitch)
* App menu and powermenu - [rofi](https://github.com/davatorium/rofi)
* Shell - [fish](https://github.com/fish-shell/fish-shell)

# How to install
1. Clone this repo:
    ```shell
    git clone https://github.com/vidy007/dotfiles.git
    ```
2. `cd` to cloned repo
3. Copy all the folders in your `.config` folder
4. Enjoy
